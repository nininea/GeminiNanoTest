# Android Image Description App

This is a simple **Android application** built using **Jetpack Compose** that demonstrates how to pick an image from the device's gallery and generate a **description for the selected image** using **ML Kit's GenAI API**.

---

## Features

- **Jetpack Compose**: Modern declarative UI framework for building clean and reactive user interfaces.
- **Image Picker**: Select an image from the device's gallery using the Android system picker (`ActivityResultContracts.GetContent()`).
- **ML Kit's GenAI API**: Generate natural language descriptions of the selected image with the power of ML Kit's Generative AI foundation models.
- **Fast and Private**: Leverages on-device processing for faster execution and enhanced user privacy.
